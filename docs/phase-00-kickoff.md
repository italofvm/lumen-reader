# Phase 0: Kickoff & Initial Documentation

## Objetivos
- Alinhamento de expectativas e requisitos.
- Definição do escopo inicial (MVP).
- Estruturação documental do projeto.

## Resumo do Projeto
**Lumen Reader** é um aplicativo de leitura de ebooks focado em privacidade, performance offline e integração inteligente com IA.

## Requisitos Iniciais
### Funcionais
- Importação de arquivos PDF e EPUB.
- Leitura offline de livros.
- Gerenciamento de progresso de leitura.
- Favoritos e anotações.
- Personalização de tema e fontes.
- Explicação de texto via Gemini API.

### Não Funcionais
- Arquitetura Limpa (Clean Architecture).
- Persistência local com Hive.
- Foco em leitura sem distrações.
- Expansibilidade para iOS e macOS.

## Escopo do MVP
- Importação local.
- Leitor PDF/EPUB básico.
- Configurações de tema.
- Integração inicial Gemini (explicar texto).
